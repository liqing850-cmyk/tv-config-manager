# tv-config-manager

轻量的 TV 配置管理面板（React + Vite + Zustand）。

快速开始

1. 安装依赖

```bash
npm install
```

2. 启动开发服务器

```bash
npm run dev
```

开发提示

- 默认管理员密码：`admin`（登录后可管理 profile、视频源、直播源及 TVBox 配置）。
- 如果在远程容器或 Codespaces 中无法直接访问，请将端口 `5173` 转发为公开，或使用 `npx localtunnel --port 5173` 暴露临时地址。

构建

```bash
npm run build
```
